HOW TO RUN:

Compile and run Server.java
(to run the server process)

Compile and run Main.java
(to run the client process)




ABOUT THE APPLICATIONS:

Multiple Main applications can be compiled
and run while one Server application
is running. This can be done over
local networks and online.

-No IP prompt has been implemented yet,
however you can change the string
"localhost" to an IP address to connect
to in the file helpers.StateManager's
connectIP() method.

Only one Server application can be running
on a specific machine in the current iteration,
and port 34956 must be open.
